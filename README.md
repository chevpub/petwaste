## ğŸŒ Deployment to GitHub Pages

### Option 1: Using GitHub Website

1. Create a new repository on GitHub
2. Name it something like `doodie-soldiers-website` or `your-business-website`
3. Upload all files (index.html, styles.css, script.js, README.md, and assets folder)
4. Go to repository Settings
5. Scroll to "Pages" section
6. Under "Source", select "main" branch
7. Click "Save"
8. Your site will be live at: `https://your-username.github.io/repository-name/`

### Option 2: Using Git Command Line

```bash
# Initialize git repository
git init

# Add all files
git add .

# Commit files
git commit -m "Initial commit - Doodie Soldiers website"

# Add remote repository (replace with your repo URL)
git remote add origin https://github.com/your-username/your-repo-name.git

# Push to GitHub
git push -u origin main
```

Then follow steps 4-8 from Option 1 to enable GitHub Pages.

## ğŸ¨ Customization Guide

### Colors
The color scheme uses CSS variables defined in `styles.css`:
```css
--primary-red: #DC143C;
--primary-blue: #003DA5;
--dark-blue: #001F54;
--white: #FFFFFF;
```

To change colors, simply update these values in the `:root` section.

### Fonts
Current fonts:
- **Headings**: Bebas Neue (display font)
- **Body**: Poppins (sans-serif)

To change fonts, update the Google Fonts import in `index.html` and the font-family declarations in `styles.css`.

### Content Sections
All content can be edited directly in `index.html`:
- Hero text and CTAs
- Service plans and pricing
- About section text
- Testimonials
- FAQ questions
- Contact information
